# Talk_to_my_document
**Objective**: This python code demonstrates how to build a Retrieval-Augmented Generation (RAG) chatbot using Langchain framework. It takes as input a text document that outlines employee leave policy. It chunks and builds a FAISS vector database. A Gemini LLM then uses this database to answer user questions.

**User input**: A question about employee leave policy document.

**Output**: Response from AI assistant addressing user’s question.

**.env file** contains API key, embeddings model name and LLM model name.
